Welcome to the NSBB server directory. This is one of several NSBB based sites that are meant to help lead you through certain aspects of pokemon go such as raids and shinies. This particular section of the site is for other sites. Pokemon go is a massive game and to put all its information into sites like this would be an impossible feat, however there are so many more sites out there. Each site has in depth statistics, information, research and so much more done to ensure an easy going and helpful experience. On the left hand menu, there are a few buttons that will take you to a variety of different sites. Some of these sites include serebii, which is useful for deep dives into events, bulbapedia which is a massive site for all things pokemon, leekduck which is helpful for raids and leaks for upcoming events and information, and finally a directory to the niagara region discord server. Ambassadored by an official...ambassador. Supported by the company behind pokemon go themselves. These sites are all very helpful for getting a much more in depth understanding to many more aspects of pokemon go. So please enjoy taking a look at them! :D
